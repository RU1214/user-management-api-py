# Setup Application
```
$ git clone https://github.com/RU1214/user-management-api-py.git
$ cd user-management-api-py
$ python3 -m venv venv
$ source venv/bin/activate
$ pip install flask
```

# Run the Application
`./bin/runApp.sh`
